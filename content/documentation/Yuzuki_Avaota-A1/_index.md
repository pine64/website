The **Yuzuki Avaota-A1** is an ARM64 based Single Board Computer powered by an Allwinner A527 Octa-Core ARM Cortex-A55 64-Bit CPU @2.0GHz, ARM Mali G56 GPU and RISC-V XuanTie E906 CPU@200MHz. It provides an onboard eMMC, MicroSD Card slot, PCI-e, Pi-2 Bus, USB 3.0, and many other peripheral interfaces for makers to integrate with sensors and other devices.
